# منهج التفسير التجاري الحذر

## Funnel الأدلة

يبدأ المنهج من 3,423 مرشح Phase 4، ثم يضيف Bootstrap stability وmetric intervals وmisleading flags وsubsumption. ينتج 590 Tier A و2,281 Tier B و549 Tier C و3 Tier D. القواعد النهائية المؤهلة للتفسير هي 2,871 Tier A/B stable nonredundant rule؛ لا يعني التأهيل أنها مثبتة تجارياً.

## التصنيف

- Tier A بطول كلي≥3: Product bundle candidate.
- Tier A one-to-one مع count≥180: Co-display candidate.
- Tier A one-to-one دون ذلك: Cross-sell candidate.
- Tier B: Promotional test candidate.
- Tier C: Monitor for more evidence.
- Tier D: Reject from commercial interpretation.

ينتج التصنيف 515 bundle و58 co-display و17 cross-sell و2,281 promotional test و549 monitor و3 reject. يحتفظ كل صف بـ`rule_key` والأكواد والأوصاف وSupport count وConfidence وLift وpresence وtier والأعلام.

## اللغة المسموحة والتحقق

تستخدم الصياغة candidate وpossible وsupported association وrequires business testing. لا يدعي المشروع زيادة الإيراد أو السببية أو ضمان الشراء أو أثر رفوف مثبت. تقترح الخطوات مراجعة توافق المنتجات ثم A/B test أو controlled merchandising experiment أو holdout زمني أو جمع بيانات إضافية.

## Traceability والحدود

يمكن العودة من كل action إلى قاعدة الجودة ثم observations والـBootstrap seed metadata. لا تستبدل الفئة المقاييس الأصلية. يتأثر التصنيف بحدود المشروع وبـ`max_len=3` و20 resamples وسوق UK وسياسة التنظيف. لم تُجر تجربة تجارية، لذلك تبقى النتائج فرضيات قابلة للاختبار فقط.
